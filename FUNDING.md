## What Funding Supports

Your support directly funds:

- 🔬 **Accuracy Improvements** - Validation against reference software, testing
- 📚 **Documentation** - Guides, examples, API references
- 🛠️ **New Features** - Advanced calculations (progressions, returns, etc.)
- 🎨 **Web Interface** - User-friendly UI for birth chart calculation
- 🔌 **Integrations** - APIs, database support, third-party connections
- 🌍 **Community** - Time spent supporting contributors and answering questions
- ⚡ **Performance** - Optimization and scaling improvements
- 📦 **Maintenance** - Keeping dependencies current and project healthy

## Transparent Use of Funds

All financial support is used directly for:
1. **Development time** - Implementing features, testing, validation
2. **Infrastructure** - Hosting, CDN, databases if needed
3. **Documentation** - Professional guides and examples
4. **Community** - Tools to support contributors

We do not:
- Charge for open-source code (MIT license)
- Use funds for salaries (volunteer project)
- Hide how funds are used
