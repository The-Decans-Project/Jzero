# Mission

I couldn't find what I wanted, so I built it.

Now I want to give back to the community.

## What This Is

Jzero is an open-source framework for astrology calculations. It's built with accuracy first and Swiss Ephemeris integration. It's meant to be:

- **Understandable** - Clean code you can learn from
- **Accurate** - Real astronomical calculations (±0.0001°)
- **Open** - MIT licensed, fully transparent
- **Community-driven** - Easy to extend and improve

## Why It Exists

There wasn't a good, simple, open astrology calculator that I could point people to. Most are closed source or locked behind paywalls. This needed to exist.

## How You Can Help

- Use it and provide feedback
- Report issues and suggest improvements

This is a community project. It's only as good as the people building it together.
