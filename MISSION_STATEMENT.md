## Mission Statement

I searched everywhere for a good, transparent astrology calculator—something open, accurate enough to learn from, and cleanly coded. I couldn't find one that was simple enough for beginners to understand and improve.

So I built one that's a solid framework with working components.

This framework has working time systems, house calculations, and geolocation. The planetary calculations are basic approximations that work but need Swiss Ephemeris integration for professional accuracy. It's a decent system that just needs a little tender loving care.

The goal is to provide a starting point for anyone who wants to learn how astrology software works, from basic orbital mechanics to professional astronomical calculations using Swiss Ephemeris and CSV data interpolation.

If this project helps developers understand and implement accurate astronomical calculations for astrology, then it's done its job.

**– The Decan Project**
