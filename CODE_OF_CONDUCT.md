# Code of Conduct

## Our Pledge

Be respectful. Include others. Assume good intent.

## Our Standards

**Do:**
- Be kind and respectful
- Welcome different perspectives
- Give constructive feedback
- Help each other learn
- Have empathy for others

**Don't:**
- Harass or insult anyone
- Make derogatory comments
- Post others' private information
- Troll or fight
- Make it unwelcoming

## Scope

This applies to:
- GitHub issues and pull requests
- Comments and discussions
- Code reviews
- Community interactions

## Reporting

If someone behaves disrespectfully:
- Let a maintainer know privately
- We'll address it fairly

Everything else: let's just be good to each other.
- Project documentation
- Community events and meetings

## Enforcement

Instances of unacceptable behavior may be reported by contacting the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

**Reporting**: Contact maintainers through:
- Email: [maintainer email - to be added]
- GitHub Issue: Create a private security issue

All reports will be kept confidential.

## Consequences

Project maintainers who violate the Code of Conduct will be held to the same standard. Consequences may include:
1. **Warning**: Private message explaining the violation
2. **Temporary Mute**: Restrictions on commenting/PR participation (24-48 hours)
3. **Ban**: Removal from the project and community

## Community Impact

We believe in restoration over punishment. Contributors who violate this code are given opportunities to learn and do better.
