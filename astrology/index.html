🚀 Starter Astrology Framework

🎯 What You Get

This framework provides all the essential building blocks for astrology applications without being a complete software. It's designed so developers can build upon it and create their own astrology tools.

📊 Ephemeris Calculations

Basic planetary position calculations using simplified Keplerian elements. Replace with Swiss Ephemeris for production accuracy.

🏠 House Systems

Multiple house system implementations: Equal, Porphyry, Whole Sign, and basic Placidus. Framework for adding more systems.

🌟 Transits

Current planetary positions and transit aspects. Find upcoming significant transits and void-of-course Moon periods.

💕 Synastry

Chart comparison for relationships. Calculate inter-aspects, composite charts, and house overlays.

⏳ Progressions

Secondary progressions, solar arc directions, and solar returns. Framework for developmental astrology.

🕐 Time Utilities

Julian day calculations, sidereal time, timezone conversions, and date parsing. Essential for accurate astrology.

📍 Location Database

City lookup with coordinates and timezones. Framework for expanding with more locations.

💾 Chart Database

Basic CRUD operations for storing and retrieving charts. In-memory storage - extend with real database.

💡 How to Use

Each module provides the core calculations and data structures. Here's how to get started:

1. Calculate a Natal Chart

// Create a chart var chart = new Chart({ name: 'John Doe', birthdate: '1990-05-15T14:30:00Z', location: 'New York, USA' }); // Calculate planetary positions (you implement the actual calculations) chart[Sun] = { longitude: 245.5, sign: Taurus }; chart[Moon] = { longitude: 120.3, sign: Virgo }; // ... add other planets // Calculate aspects var aspects = chart.getAllAspects();

2. Calculate Transits

// Get current transits for a natal chart var transits = Transits.calculateTransits(natalChart, new Date()); // Find upcoming significant transits var upcoming = Transits.findUpcomingTransits(natalChart, 30); // Next 30 days

3. Synastry Analysis

// Compare two charts var synastry = Synastry.calculateSynastry(chart1, chart2); console.log('Compatibility score:', synastry.scores.overall); console.log('Summary:', synastry.summary);

4. Progressions

// Calculate secondary progression var progressed = Progressions.calculateSecondaryProgression(natalChart, 25); // Age 25 // Calculate solar return var solarReturn = Progressions.calculateSolarReturn(natalChart, 2024);

🔧 Extend the Framework

The framework is designed to be extended. Here are key areas to enhance:

  • Ephemeris: Replace simplified calculations with Swiss Ephemeris or astronomical libraries
  • House Systems: Add Koch, Campanus, Regiomontanus, and other house systems
  • Fixed Stars: Add calculations for important fixed stars
  • Asteroids: Include Chiron, Juno, Vesta, and other asteroids
  • Database: Replace in-memory storage with SQLite, MongoDB, or cloud database
  • UI: Add chart wheels, aspect grids, and interactive displays
  • Reports: Create more specialized reports (career, health, relationships)
  • APIs: Add REST endpoints for web applications
